# CASINO-RADAR

Simple GitHub Pages deployment. Put `funnel.html` and `landing.html` in the repo root.

To publish:

git init
git add .
git commit -m "Initial CASINO-RADAR site"
git branch -M main
git remote add origin git@github.com:<your-username>/CASINO-RADAR.git
git push -u origin main

Then enable GitHub Pages in repo Settings → Pages → choose `main` branch / root.
Your site will be available at: `https://<your-username>.github.io/CASINO-RADAR/funnel.html`
